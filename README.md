# Personal Productivity Tracker

A lightweight web app to plan daily intentions, map time blocks, and capture reflections.

Live site:
- https://lalan-lahera.github.io/personal-productivity-tracker/

## Features
- Add tasks with duration, energy level, and priority
- Mark tasks done/undo and remove them
- Plan time blocks by type (focus, meeting, recovery)
- Live stats: focus score, deep work, recovery, intention count
- Daily reflection notes with save timestamp

## Tech Stack
- HTML
- CSS
- JavaScript

## Local Run
Open `index.html` in any modern browser.
